# Copilot Task Prompt Template

When assisting with this project assume:

- audio files are long-form environmental recordings
- metadata is stored in CSV tables
- audio is chunked into 1-hour segments
- listening copies are Opus files
- storage uses Cloudflare R2
- website hosted via Cloudflare Pages

When writing code:

- keep dependencies minimal
- prioritize clarity and reproducibility
- prefer simple scripts over complex frameworks
